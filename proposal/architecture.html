Known Gaps — Remediation Scope

🔐
No authentication or authorization
All endpoints are publicly accessible. CORS is configured to allow all origins (*). Production deployment requires auth middleware and origin restriction. Outside current SOW; flagged for Meridian IT.
💾
No database — data lost on restart
All writes (e.g., purchase order creation) are in-memory only. Restarting the server resets to the JSON file state. Suitable for demo; requires a persistent store (PostgreSQL, etc.) for production.
📊
Reports module defects (R1)
Filter wiring incomplete at handoff — some combinations produce incorrect results. Console errors present. Under active remediation in this engagement.
🧪
No automated test coverage delivered
Previous vendor delivered zero automated tests. E2E test scaffolding (Playwright) and backend test suite (pytest) are being added as R3 deliverables in this engagement.
🌐
Partial i18n coverage
English and Japanese locale files exist but some UI strings are hardcoded in English. Currency conversion uses a hardcoded exchange rate (1 USD = 150 JPY). Tracked under D2.
